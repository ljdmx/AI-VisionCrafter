### 1. 首次设置：API 密钥
首次启动应用时，系统会弹出一个窗口，要求您输入 Google Gemini API 密钥。

*   **为什么需要密钥？** 本应用的所有 AI 功能都由 Google Gemini API 驱动。您需要提供自己的密钥来授权应用调用这些服务。
*   **如何获取密钥？** 您可以访问 [Google AI Studio](https://aistudio.google.com/app/apikey) 免费创建一个 API 密钥。
*   **安全性**：您的 API 密钥只会保存在您本地浏览器的存储中，不会被上传到任何服务器，确保了您的密钥安全。

输入您的密钥并点击保存后，您就可以开始您的创意之旅了。

### 2. 简介

AI VisionCrafter 是一个集成化的 AI 视觉创作平台，旨在为您提供前所未有的创作自由度和控制力。它包含三个核心模块，满足您不同的创意需求：

*   **文生图**：将您的文字描述转化为独一无二的精美图像。
*   **图像编辑**：通过自然语言指令，对现有图片进行智能修改、优化或风格转换。
*   **图像融合**：提取一张图的内容和另一张图的艺术风格，创造全新的艺术作品。

### 3. 开始创作：主页

打开应用后，您会看到主页和三个创作模式选项。

*   点击 **图像编辑** 卡片进入图片修改模式。
*   点击 **文生图** 卡片从零开始创作。
*   点击 **图像融合** 卡片来混合两张图片。

选择任意模式即可开始您的创意之旅。在每个创作界面的左上角，都有 **返回主页** 按钮，方便您随时切换模式。

### 4. 创作模式详解

#### 4.1 文生图 (Text-to-Image)

此模式用于将您的想象力通过文字生成图像。

**工作流程:**

1.  **描述您的画面**: 在最大的文本框中，详细输入您想看到的画面内容。描述越具体、越生动，生成的效果越好。
2.  **魔术提示词 (可选)**: 如果您只有一个简单的想法，可以先输入，然后点击 **✨ 魔术提示词** 按钮。AI 将自动将您的想法扩展成一段充满细节、专业级的提示词，极大地提升出图质量。
3.  **参考角色 (可选)**: 如果您希望生成的图像中包含一个具有特定外观的角色，可以在此上传一张参考图。AI 会分析角色的特征（如发型、服装）并将其融入到您的新创作中。上传后，可以点击图片右上角的 `x` 来移除它。
4.  **负向提示词 (可选)**: 在此输入您 **不希望** 在画面中看到的内容，例如“模糊”、“低画质”、“多余的手指”等。
5.  **宽高比**: 选择您想要的图像比例。
    *   **方案 (方形)**: 1:1 比例，适用于社交媒体头像等。
    *   **横向 (16:9)**: 宽屏比例，适用于壁纸、封面图等。
    *   **纵向 (9:16)**: 竖屏比例，适用于手机壁纸、短视频素材等。
6.  **艺术风格 (可选)**: 点击浏览丰富的预设艺术风格。选择一个后，AI 会自动应用该风格的关键词，生成相应风格的图像。再次点击可取消选择。
7.  **专业参数 (可选)**:
    *   **种子 (Seed)**: 这是一个数字，用于控制图像生成的初始随机性。使用相同的提示词和相同的种子，可以生成非常相似的图片。这对于微调和复现结果非常有用。您可以手动输入一个数字，或点击右侧的 **骰子图标** 来随机生成一个。
8.  **生成图像**: 完成设置后，点击 **✨ 生成图像** 按钮。请稍候片刻，您的作品就会在左侧区域呈现。
9.  **下载**: 生成的图片右下角会出现 **下载图像** 按钮，方便您保存作品。

#### 4.2 图像编辑 (Image Edit)

此模式用于修改您已有的图片。

**工作流程:**

1.  **上传图片**: 首先，从您的设备上传一张图片，或直接将图片文件拖拽到上传区域。支持 JPG 和 PNG 格式，最大 5MB。
2.  **修改指令**: 在主文本框中输入您想如何修改图片。例如“把天空换成星空”或“给人物戴上一顶帽子”。您同样可以使用 **🪄 魔术提示词** 按钮来优化您的指令。
3.  **参考角色/风格 (可选)**: 上传一张参考图，AI 会尝试将参考图的角色特征或艺术风格应用到您的修改中。
4.  **局部编辑 (可选)**:
    *   开启 **启用蒙版** 开关。
    *   您的鼠标会变成一个笔刷。在图片上 **涂抹您想要修改的区域**。
    *   您可以使用 **笔刷大小** 滑块来调整笔刷粗细。
    *   如果涂抹错误，可以点击 **清除蒙版**。
    *   启用后，您的修改指令将 **只对涂抹的区域生效**，图片的其他部分将保持不变。
5.  **专业参数 (可选)**: 您同样可以在此设置 **负向提示词** 和 **种子 (Seed)** 来获得更精确的控制。
6.  **应用修改**: 点击 **🪄 应用修改** 按钮。AI 会根据您的指令和设置处理图片。
7.  **后续操作**:
    *   **下载图像**: 在图片下方的工具栏中，点击下载按钮保存您编辑后的作品。
    *   **更换图片**: 如果想换一张底图但保留当前的指令和参数，可以点击 **更换图片** 按钮。

---

#### 4.3 图像融合 (Image Remix)

此模式将一张图片的内容与另一张图片的风格相融合。

**工作流程:**

1.  **上传内容图**: 上传一张您想保留其 **主体内容和构图** 的图片。
2.  **上传风格图**: 上传一张您想借鉴其 **色彩、笔触、光影等艺术风格** 的图片。
3.  **额外指令 (可选)**: 输入额外的文字指令来进一步引导融合效果。
4.  **负向提示词 (可选)**: 输入不希望在最终结果中出现的内容。
5.  **宽高比**: 选择最终输出图像的比例。
6.  **专业参数 (可选)**: 使用 **种子 (Seed)** 来控制生成结果的随机性。
7.  **融合图像**: 点击 **🎨 融合图像** 按钮。AI 会开始分析并融合两张图片。
8.  **下载**: 在生成的图片右下角点击 **下载图像** 按钮保存您的创意融合作品。

### 5. 通用技巧

*   **文件大小**: 所有图片上传功能都支持最大 5MB 的 JPG 或 PNG 文件。如果文件过大，系统会弹出提示。
*   **清晰的指令**: 无论在哪种模式下，清晰、具体、富有描述性的文字指令都是获得高质量结果的关键。
*   **不断尝试**: AI 创作充满了可能性。尝试不同的提示词、风格和参数组合，您可能会收获意想不到的惊喜！

祝您创作愉快！
